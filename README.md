# @amlplugins/sendgrid-suppression

Native `@sendgrid/client` SDK re-exported under the `@amlplugins` namespace.

> SendGrid Suppressions — bounces, blocks, spam reports, invalid emails, unsubscribe groups.

## Install

```bash
npm install @amlplugins/sendgrid-suppression
```

## Use

```ts
import { /* named exports from @sendgrid/client */ } from "@amlplugins/sendgrid-suppression";
// or
import sdk from "@amlplugins/sendgrid-suppression";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
